## Opis projektu
Projekt zawiera implementację dwóch zadań:
1. Model drzew liściastych i iglastych plus testy jednostkowe
2. Usługa REST do zarządzania zadaniami zespołu z testami intergacyjnymi

## Technologia
- Java 21
- Spring Boot (wersja 3.x)
- Maven
- H2 Database
- Lombok
- Spring Data JPA
